Это — третья миссия из серии про игру риичи-маджонг. Эта миссия сложнее предыдущих, так что будьте готовы.

В этот раз ваша задача — определить, какое наименьшее число тайлов в руке надо заменить, чтобы сделать ее выигрыишной. Иными словами, нужно определить ее, так называемое, число шантен. Помните, что существует не только стандартный паттерн руки "четыре сета и пара", но и читойцу, и кокуши-мусо.

Для простоты возможность существования открытых сетов опущена.

Вот объясняющие примеры:

В следующей руке нужно заменить только один тайл ("темпай"):

['p4', 'p5', 'p6', 'm1', 'm1', 'm1', 's2', 's3', 's4', 's5', 'we', 'we', 'dw', 'dw',]

А в этой — три тайла:

['p4', 'p5', 'p6', 'm1', 'm1', 'm1', 's2', 's4', 's6', 'we', 'ws', 'dw', 'dw']

Input data: Один аргумент — список строк, кодирующих тайлы.

Output data: Переменная типа bool — ответ на вопрос "готова ли рука?".

Example: 

    riichi_mahjong_win_hand(['m1', 'm2', 'm3', 'm5', 'm5', 'p2', 'p3', 'p4', 'p7', 'p8', 'p9'], ['s456']) == True
